# HostFree Session Sync

**Удобный способ использовать автопродление HostFree без ввода пароля от hostgta.ru**

Chrome-расширение, которое забирает сессию (cookies) с hostgta.ru / hostgta.com и позволяет быстро импортировать её в [HostFree](https://hostfree.lovable.app).

## Зачем это нужно?

- Автопродление серверов работает **без хранения твоего пароля**
- Не нужно каждый раз логиниться заново
- Полная приватность — логин/пароль никуда не отправляются

## Как пользоваться

1. Установи расширение (пока вручную — как unpacked)
2. Зайди на **hostgta.ru** и войди в аккаунт
3. Нажми на иконку расширения → **"Скопировать сессию hostgta"**
4. Перейди в [HostFree → Импорт сессии](https://hostfree.lovable.app/dashboard)
5. Вставь данные из буфера

Готово. Сессия привязана.

## Установка (Developer mode)

1. Скачай репозиторий
2. Открой `chrome://extensions/`
3. Включи **"Режим разработчика"**
4. Нажми **"Загрузить распакованное расширение"** и выбери папку с файлами

## Файлы расширения

- `manifest.json` — манифест
- `popup.html` — интерфейс
- `popup.js` — логика получения cookies

## Технологии

- Manifest V3
- Chrome Cookies API
- Clipboard API
